# Руководство по запуску Easy-ice (Android)

Проект **Easy-ice** написан на **.NET MAUI** (не на Java/Kotlin), поэтому Android Studio используется здесь только для установки **Android SDK** и **Эмулятора**. Код редактируется и запускается через Visual Studio (или VS Code).

## 1. Настройка Android Environment

### Шаг 1: Запуск Android Studio
1.  Запустите **Android Studio**.
2.  Если это первый запуск, следуйте **Setup Wizard**.
    *   Выберите тип установки **Standard**.
    *   Убедитесь, что галочка **Android SDK** стоит.
    *   Дождитесь загрузки и установки компонентов.

### Шаг 2: Проверка пути к SDK
1.  На экране "Welcome to Android Studio" нажмите **More Actions** (или три точки) -> **SDK Manager**.
    *   Либо, если проект уже открыт: **Tools** -> **SDK Manager**.
2.  В верхней части окна скопируйте путь из поля **Android SDK Location**.
    *   Обычно это: `C:\Users\<ВашПользователь>\AppData\Local\Android\Sdk`

### Шаг 3: Создание Эмулятора
1.  В Android Studio откройте **Device Manager** (More Actions -> Virtual Device Manager).
2.  Нажмите **Create Device**.
3.  Выберите телефон (например, **Pixel 5**).
4.  Выберите образ системы (рекомендуется **API 33** или **API 34**). Скачайте его, если нужно.
5.  Нажмите **Finish**.
6.  **Запустите эмулятор** кнопкой "Play" (треугольник).

## 2. Настройка Windows (Переменные среды)

Чтобы `dotnet` увидел SDK, нужно добавить переменную среды.

1.  Нажмите `Win + R`, введите `sysdm.cpl` и нажмите Enter.
2.  Перейдите на вкладку **Дополнительно** -> **Переменные среды**.
3.  В разделе **Системные переменные** нажмите **Создать**:
    *   **Имя переменной**: `ANDROID_HOME`
    *   **Значение переменной**: *Вставьте путь к SDK, который скопировали ранее* (например, `C:\Users\aknik\AppData\Local\Android\Sdk`).
4.  Нажмите **ОК** во всех окнах.
5.  **Перезапустите терминал / VS Code**, чтобы изменения вступили в силу.

## 3. Запуск Проекта

После настройки SDK и запуска эмулятора, выполните команду в терминале (в папке проекта):

```powershell
dotnet build -t:Run -f net8.0-android src/Frontend/EasyIce.Mobile/EasyIce.Mobile.csproj
```

Или, если вы используете **VS Code**:
1.  Установите расширение **.NET MAUI**.
2.  Откройте палитру команд (`Ctrl+Shift+P`).
3.  Выберите **.NET MAUI: Pick Android Device** -> Выберите ваш эмулятор.
4.  Нажмите F5 для запуска.
